# Messenger App

## Live Demo
https://messenger-riddhika.vercel.app


## Tech Stack
- Next.js 16
- TypeScript
- Prisma
- MongoDB Atlas
- NextAuth.js
- Pusher
- Cloudinary
- Tailwind CSS
- React Hook Form


## Features
- Secure authentication (Google, GitHub, Credentials)
- Real-time messaging using Pusher
- Online/offline user status
- Image sharing with Cloudinary
- Group conversations
- Read receipts
- Responsive UI

## Screenshots
Application screenshots can be found in the `screenshots/` directory.

## Installation
git clone https://github.com/Riddhikapaliwal/Messenger-app.git
cd messenger-clone
npm install
npm run dev
